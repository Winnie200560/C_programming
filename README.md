## Задание 1.
### Текст задачи
Разработать класс с тремя целочисленными полями. Создать конструктор копирования.
Разработать метод, вычисляющий максимальное из полей. Перегрузить метод ToString() для
формирования строки из полей класса. Реализовать дочерний класс (его содержание предложить
самостоятельно и описать в решении: какой содержательный смысл имеют поля; реализовать
конструкторы; предложить и реализовать 2-3 метода). Протестировать все конструкторы и
другие методы базового и дочернего классов.
### Алгоритм решения
1. Пользователь вводит данные транспортного средства: название, тип, расстояние, время и скорость.
2. Далее создаётся объект класса Transport, при этом через конструктор вызывается базовый класс Motion, где сохраняются расстояние, время и скорость.
3. Класс Motion содержит метод нахождения максимального значения.
4. Класс Transport добавляет поля (название и тип транспорта) и методы(вычисление средней скорости, проверка превышения скорости)
5. Результаты выводятся на экран.

## Задание 2.
### Текст задачи
Название класса: Point. 
Поля: double x, double y (координаты точки)
Методы: Вычислить расстояние от одной точки до другой. Результат должен быть типа double.
### Алгоритм решения
1. Пользователь вводит координаты двух точек на плоскости: x1, y1 для первой точки и x2, y2 для второй точки.
2. Далее выполняется вычисление разности координат по осям. На основе полученных значений определяется расстояние между точками по формуле Евклидова расстояния (т.е результат вычисляется как квадратный корень из суммы квадратов разностей координат).
3. Полученное значение выводится на экран.

## Задание 3.
### Текст задачи
Унарные операции:
++ увеличить координаты xна 1,
-- уменьшение координаты х на 1.
Операции приведения типа:
int (явная) – результатом является целая часть координаты х;
double (неявная) – результатом является координата y.
Бинарные операции:
+ Point p – вычисляется расстояние до точки p. Результатом должно быть
вещественное число.
+ целое число (лево- и правосторонние операции). Увеличивается
координата х, результатом является объект типа Point.
### Алгоритм решения
1. Доплняется задача №2 с помощью перегрузки операторов.
2. В классе реализуются унарные операции: оператор ++ увеличивает координату x точки на 1, оператор -- уменьшает координату x на 1.
3. Также реализуются операции приведения типов: явное приведение типа int, при котором результатом является целая часть координаты x текущего объекта, и неявное приведение типа double, при котором результатом является значение координаты y.
4. Перегружаются бинарные операции. Оператор сложения между двумя объектами Point используется для вычисления расстояния между точками. Также перегружается оператор сложения между объектом Point и целым числом, при котором увеличивается координата x текущей точки. Операция поддерживается в обоих направлениях (Point + int и int + Point).
5. Результаты всех операций выводятся на экран.
