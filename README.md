## Задание 1.
### Текст задачи
Разработать класс с тремя целочисленными полями. Создать конструктор копирования.
Разработать метод, вычисляющий максимальное из полей. Перегрузить метод ToString() для
формирования строки из полей класса. Реализовать дочерний класс (его содержание предложить
самостоятельно и описать в решении: какой содержательный смысл имеют поля; реализовать
конструкторы; предложить и реализовать 2-3 метода). Протестировать все конструкторы и
другие методы базового и дочернего классов.
### Алгоритм решения
1. Базовый класс - Motion. Три целочисленных поля: расстояние, время, скорость (объявлены как private).
2. Реализован конструктор без параметров (инициализирует поля значениями по умолчанию: расстояние = 0, время = 1, скорость = 0), конструктор с параметрами (принимает значения расстояния, времени и скорости), конструктор копирования (создает новый объект на основе существующего).
3. Для каждого поля реализованы свойства Distance, Time, Speed. Они содержат проверку данных (расстояние, скорость не могут быть отрицительными, время < 0).
4. Методы. Метод поиска максимального значения из трех полей, реализован последовательным сравнением. Переопределение метода ToString - формирует строку для вывода значений.
5. Дочерник класс - Transport, наследуется от Motion. Два поля класса: название и тип транспорта.
6. Реализованы конструкторы без параметов (вызывает базовый конструктор (base) и задаёт значения по умолчанию), конструктор с параметрами(принимает название, тип, расстояние, скорость и время) и конструктор копирования (копирует данные объекта, включая данные из базового класса).
7. Для каждого поля реализованы свойства Name, Type, которые проверяют является ли вводимая строка пустой.
8. Методы. Метод AverageSpeed() вычисляющий средняюю скорость, метод IsSpeeding(limit) проверяет превыщение скорости, переопределение метода ToString.
9. В main тестируются все конструкторы и методы.
10. Результаты выводятся на экран.

## Задание 2 и 3.
### Текст задачи
Название класса: Point. 
Поля: double x, double y (координаты точки)
Методы: Вычислить расстояние от одной точки до другой. Результат должен быть типа double.
Унарные операции:
++ увеличить координаты xна 1,
-- уменьшение координаты х на 1.
Операции приведения типа:
int (явная) – результатом является целая часть координаты х;
double (неявная) – результатом является координата y.
Бинарные операции:
+ Point p – вычисляется расстояние до точки p. Результатом должно быть
вещественное число.
+ целое число (лево- и правосторонние операции). Увеличивается
координата х, результатом является объект типа Point.

### Алгоритм решения
1. Класс Point с двумя полями: x и y, которые описывают точку.
2. Реализованы конструктор по умолчанию (инициализируютс нулями), конструктор с параметрами (принимает значения точки), конструктор копирования (содает копию точки).
3. Реализованы свойства X и Y, которые обеспечивают доступ к закрытым полям _x и _y.
4. Реализован метод DistanceTo, который вычисляет расстояние между двумя точками по формуле. Метод ToString() переопределён для вывода координат точки.
5. Так же перегружены операторы: ++ и -- — изменение координаты х на 1, + — вычисление расстояния между точками, + с числом — сдвиг точки по оси X, приведение типов (int и double).
6. В main тестируются все конструкторы и методы.
7. Результаты выводятся на экран.
